# Project 10: Text Manipulator

A simple text manipulation tool that demonstrates JavaScript functions and DOM manipulation.

## Concepts Covered:
- JavaScript Functions
- DOM Manipulation
- String Methods (toUpperCase, toLowerCase, charAt)
- Event Listeners
- CSS Style Manipulation

## Features:
- Convert text to uppercase
- Convert text to lowercase
- Capitalize first letter
- Change text color (red, blue, green)
- Increase/decrease font size
- Reset all changes
- Real-time input updates

## How to Use:
1. Open `index.html` in a web browser
2. Type text in the input field or use the default text
3. Click buttons to manipulate the text
4. Watch the display update in real-time!
